Formula Indicators

Formula Indicators use existing indicators to calculate new metrics and provide deeper insights into business process performance. Unlike Automated Indicators, Formula Indicators do not require an Indicator Source.

Configuration and Formula Building
  • Navigate to Platform Analytics Administration > Indicators > Formula Indicators.
  • Define the Formula Text using JavaScript syntax with references to existing indicators and breakdowns:
    • Indicators: [[ Indicator ]]
    • Breakdowns: [[ Indicator > Breakdown = Element ]]
  • You can manually type the formula or use the Browse for an Indicator UI action to select indicators and breakdowns.
  • Use the Apply time series to result option to control whether time series calculations apply to each component or only the final result.
Troubleshooting and Best Practices
  • If a formula component is null or the denominator is 0, the formula evaluates to null (“No Score”).
  • Enable Allow formula component to be NULL to handle null values in conditional statements.
  • Example conditional handling: 
    if (denominator == null || denominator == 0) {
    return -1;
} else {
    return numerator / denominator;
}
  • Formulas are executed server-side and results are not stored in tables.
  • Complex formulas may impact system performance.

Manual Indicators

Manual Indicators track trends in data or processes that are managed outside of ServiceNow. Data is entered manually or imported directly into the scores tables.

Why Use Manual Indicators?
  • Summarize information from external systems that is sensitive or unavailable in ServiceNow.
  • Allow users to track metrics that are not automatically collected.
Configuration of Manual Indicators
  • Manual Indicators are not based on an Indicator Source.
  • The Contributor property identifies users authorized to manually enter data via the Scoresheet.
Manual Breakdowns
  • Used to group data for Manual Indicators. Elements are entered manually and not stored in a Breakdown Source.
  • Steps to create a Manual Breakdown:
    1. Navigate to Platform Analytics Administration > Breakdowns > Manual Breakdowns and select New.
    2. Enter a Name and optional Description.
    3. Type in Breakdown Element values and adjust their order if needed.
    4. Associate the Breakdown with one or more Manual Indicators.
    5. Use Aggregate scores... in the Scoresheet to calculate SUM or Average of Indicator scores by Breakdown element.
    6. View the Indicator in KPI Details and edit the Scoresheet to add Breakdown Element data.

Using Formula and Manual Indicators allows you to combine calculated metrics with manually tracked data to gain a comprehensive view of process performance.
