Targets and Thresholds

Targets and Thresholds help monitor and improve business process performance by setting goals and alerting on deviations.

Targets Overview

  • Targets define quantifiable goals for Indicators, providing context and direction for process improvement.
  • Targets can be Personal (visible only to the user) or Global (visible to all users).
  • Target creation requires:
    • Target Type (For me / For everyone)
    • Target Value
    • Target Start Date
    • Target Review Date
  • Enable the Targets chart option in KPI Details to view existing Targets.

Managing Targets

  • Personal Targets: Users can create their own targets only visible to them.
  • Global Targets: Can be created only by users with roles pa_target_admin, pa_admin, or pa_power_user.
  • The Target Gap shows the difference between the Indicator Score and the Target and is color-coded based on Indicator Direction.
  • Target Color Schemes define colors for different levels of deviation. Examples include:
    • 5-color scheme: Bright Red, Dark Red, Orange, Light Green, Dark Green
    • 3-color scheme: Customizable for simpler visual guidance

Improvement over Baseline

  • Automatic Target Calculation can set targets as an improvement over baseline values.
  • Choose calculation parameters:
    • Measured as Percentage or Units
    • Average of shown scores or existing target
  • Indicator Direction determines whether improvement is added or subtracted.
  • Examples:
    • Existing target: 200, 10% improvement → New target = 180
    • Existing target: 200, 10 Units improvement → New target = 190
  • Targets can be manually adjusted after automatic calculation.

Thresholds Overview

  • Thresholds notify when an Indicator is approaching specific conditions for early detection of unusual or critical behavior.
  • Thresholds can be Personal (visible only to the owner) or Global (visible to all, with notifications sent to added users).
  • Different types of Thresholds can be defined based on business requirements.

Notifications for Targets and Thresholds

  • Notifications are sent automatically when Targets or Thresholds are reached.
  • Add users to the Users Related List in the Target or Threshold form to receive notifications.
  • Personal Thresholds/Targets automatically notify the owner if subscribed.
  • Users can manage their notification subscriptions under User Preferences → Notifications → Advanced Preferences.

Proper use of Targets and Thresholds ensures measurable goals are tracked and deviations are promptly addressed, helping drive continuous process improvement.
