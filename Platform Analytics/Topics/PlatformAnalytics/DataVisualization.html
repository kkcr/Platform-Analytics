Data Visualization and Types

New Data Visualizations are often created in-line directly from the dashboard canvas. The default visualization type is a Single Score, which can be changed in the Configuration panel.

Creating a New Visualization

  • Select Data Visualization from the Add new element menu.
  • Choose New Visualization.
  • A blank visualization is added, and the Configuration panel opens.
  • Resize and move the element on the canvas as needed.
  • Save your changes by selecting the Save button.

Types of Data Visualizations

Each visualization type is designed to display data differently. Select the right type based on the metric and story you want to present:

  • Score Visualizations - Show a single value (e.g., KPI as number/percentage).
  • Bar Visualizations - Compare scores across dimensions (Horizontal or Vertical).
  • Pie & Donut - Show parts-to-whole relationships (segments total 100%).
  • Time Series (Trends) - Emphasize data over time and changes in values.
  • Time Series (Comparisons) - Compare categorical data across sources over time.
  • Multidimensional Charts - Show relationships between multiple variables.
  • Other Visualizations - Calendars, lists, or scorecards.

Defining a Data Source

Every visualization needs a data source. The most common types are:

  • Table Data Source - Current state of records in a table (rows = records, columns = fields).
  • Indicator Data Source - Metric-driven data used for tracking KPIs and trends.

To add a source: Select + Add data source from the Configuration panel and define filters or conditions for accuracy.

Configuring Metric Aggregation

Metrics define the calculation applied to your dataset. By default, metrics are set to COUNT. Other options include:

  • Sum of a field
  • Average of a field
  • Count distinct records
  • Minimum or Maximum field value

Always provide a clear label for the metric (e.g., “Avg Reassignment”) for business readability.

Visualization and Display Settings

Once your visualization, data source, and metric are configured, adjust chart settings:

  • Header & Border - Show chart title, description, or borders.
  • Data Update - Apply refresh intervals, filters, or live updates.
  • Presentation - Adjust score size, icons, and overall chart layout.

Dashboard Filters

Filters refine data shown on a dashboard. Types include:

  • Single Select - One value from a set of choices.
  • Multiple Select - Several values at once (e.g., High + Critical).
  • Date - Calendar or time-based filtering.
  • True/False - Binary values (e.g., Active = True).

Filter sources can be defined by Table fields or Indicator Breakdowns.

Managing Data Visualizations

While editing a visualization, use the Ellipse (…) menu for management actions:

  • Add to Library - Save visualizations for reuse across dashboards.
  • Duplicate - Copy a visualization for quick modifications.
  • Share - Grant View/Edit permissions to users, groups, or roles.
  • Export - Download or email as PDF, PNG, JPEG (various formats/sizes).
  • Bookmark - Quick access under Bookmarked Visualizations.
  • Delete - Permanently remove the visualization (Admin/viz_admin required).

Note: Only visualizations saved to the Library appear in the Analytics Center for sharing and exporting.
